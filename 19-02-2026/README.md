
**Introdução**

Timeline
1.   1989 - Criação por Guido van Rossum
2.   2000 - Python 2.0 (introdução do garbage collector)
3.   2008 Python 3.0 (remoção de código obsoleto e manipulação mais consistente de strings)
4.   2010 Adoção gradual do Python 3.0 (devido a imcompatibilidade com a versão 2.0)
5.   2015 Python 3.5 adição das corrotinas assíncronas async/await
6.   2018 Python 3.7 com melhorias de desempenho (inclusão de data classes simplificando a criação de classes simples e armazenamento de dados)
7.   2020 Python 2.0 deixa de ter o suporte oficial
8.   Atualmente o Python 3.X é dominante no mercado
