## 📌 Sobre o Repositório

Este espaço é dedicado a documentar meus estudos, códigos e experimentações com treinamento e implementação de Inteligências Artificiais. O material aqui presente é fruto das aulas de Análise e Desenvolvimento de Sistemas na UMFG, além de estudos autônomos contínuos.

O fluxo de trabalho documentado neste repositório reflete a transição do aprendizado exploratório para o código estruturado:
- **Google Colab:** Utilizado para rascunhos rápidos, testes de lógica, análise de dados e treinamento inicial de modelos aproveitando o processamento em nuvem.
- **VS Code:** Utilizado para consolidar o aprendizado, refatorar o código do Colab em scripts `.py` puros, aplicar conceitos de estruturação (como Classes e Funções) e versionar o progresso.


## 🗂️ Estrutura de Estudos

Os materiais estão divididos em pastas organizadas por datas ou módulos de aula (ex: `DD-MM-yyyy`), contendo uma documentação dupla:

* **Scripts Python (`.py`):** Além dos exemplos práticos de sintaxe e lógica de IA, os próprios arquivos contêm **anotações detalhadas no código** (usando comentários `#` e docstrings `""" """`) para explicar o funcionamento técnico linha a linha.
* **Resumos Teóricos (`.md`):** Arquivos Markdown secundários dentro das pastas de aula, servindo como um guia rápido que centraliza a teoria abordada, os comandos principais e os conceitos estudados naquele dia.
